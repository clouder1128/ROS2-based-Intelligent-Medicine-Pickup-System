使用流程

1

患者咨询

患者描述症状，AI分析并生成用药建议

2

医生审批

医生审核AI建议，确保用药安全合理

3

自动配药

审批通过后，系统自动调用药房配药

4

患者取药

患者凭取药码到药房取药
