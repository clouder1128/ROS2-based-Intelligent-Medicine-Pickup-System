**第 3-4 周：药品排序/评分引擎**

```python
# agent/screening/ranking_engine.py

class RankingEngine:
    def score(self, drug, symptoms, severity, patient_info) -> float:
        """
        综合评分（0-1）：
        - 适应症匹配率: 匹配的适应症数 / 症状总数 × 0.4
        - 严重程度加权: 症状严重度与药品适应症匹配 × 0.2
        - 过敏排除: 过敏药品直接 0 分 × 0.2
        - 库存可用性: min(quantity / min_stock_level, 1) × 0.1
        - 处方适宜性: 患者年龄/体重匹配 × 0.1
        """
```

- 新建 `agent/screening/ranking_engine.py`
- 新建 `agent/screening/__init__.py`
- 编写评分测试用例

```python
# api/chat_controller.py
@chat_bp.route("/api/chat", methods=["POST"])
def chat():
    """前端患者页面的对话接口"""
    data = request.get_json()
    message = data.get("message", "")
    patient_id = data.get("patient_id", "web_user")
    agent = MedicalAgent()
    response, steps = agent.run(message, patient_id=patient_id)
    return jsonify({
        "success": True,
        "response": response,
        "steps": [
            {"tool": s.get("tool"), "type": s.get("type")}
            for s in steps if s.get("type") == "tool_call"
        ],
    })
```

```json
{
    "success": true,
    "data": { ... },
    "pagination": {
        "page": 1,
```

### 错误格式

```json
{
    "success": false,
    "data": null,
    "error": {
        "code": "DRUG_NOT_FOUND",
        "message": "未找到 ID 为 999 的药品"
    }
}
```

### 分页规范

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `page` | int | 1 | 页码，从 1 开始 |
| `limit` | int | 20 | 每页条数，最大 100 |
| `sort_by` | string | "drug_id" | 排序字段 |
| `sort_order` | string | "asc" | asc 或 desc |
