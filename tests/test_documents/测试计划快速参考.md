# 测试计划执行快速参考

### 系统概述
**项目**：ROS2智能药品拣选系统  
**范围**：AI医疗助手 + 自动化药房后端 + ROS2集成  
**目标**：确保系统功能完整、性能稳定、质量达标  

### 关键测试阶段

| 阶段 | 周期 | 主要工作 | 完成指标 |
|------|------|--------|--------|
| **单元测试** | 第1周 | 测试各个API、工具函数、权限模块 | 覆盖率>85%, P1=0 |
| **集成测试** | 第2-3周 | 测试API调用链、数据库、流程整合 | API可用性>95%, P1=0 |
| **端到端测试** | 第4周 | 测试完整业务流程（患者→医生→系统） | 流程成功率>95%, P1=0 |
| **ROS2集成** | 第4周并行 | 测试机器人消息发布、错误降级 | 消息成功率>95% |

### 立即行动清单

```bash
# 1️⃣ 搭建环境（Day 1）
cd agent_with_backend
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt pytest pytest-cov
python database/scripts/init_db.py

# 2️⃣ 验证环境（Day 2）
pytest tests/ --co  # 查看所有可用测试
pytest --version && flask --version

# 3️⃣ 启动后端（Day 3）
python main.py &  # 启动Flask服务

# 4️⃣ 执行测试（Day 4+）
pytest tests/ -v --cov=api --cov-report=html
```
