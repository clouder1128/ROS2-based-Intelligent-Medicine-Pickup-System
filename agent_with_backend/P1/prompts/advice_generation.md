# 用药建议生成

## 输入格式
{
  "drug_name": "布洛芬",
  "dosage": "成人剂量：200-400mg，每4-6小时一次",
  "patient_info": {"age": 30, "weight": 60},
  "condition_severity": "轻"
}

## 输出要求
生成专业、清晰的用药建议，包含：
1. 药物名称和剂量
2. 使用方法（饭前/饭后，频率）
3. 注意事项和禁忌
4. 何时需要就医

## 示例输出
"建议使用布洛芬缓解疼痛，剂量为200-400mg，每4-6小时一次，饭后服用以减轻胃部不适。如果疼痛持续超过3天或加重，请及时就医。"

## 注意事项
1. 基于患者年龄体重调整建议
2. 包含必要的安全警告
3. 保持专业且易于理解
