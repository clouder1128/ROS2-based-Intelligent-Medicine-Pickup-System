# 药品未找到反馈生成

## 输入格式
{
  "extracted_symptoms": ["头痛", "发热"],
  "patient_info": {"age": 30, "weight": 60}
}

## 输出要求
生成给用户的反馈消息，包含：
1. 确认识别的症状（简洁列出）
2. 说明未找到匹配药品
3. 建议看医生
4. 询问是否正确识别

## 示例输出
"根据您的描述，识别到的症状是：头痛、发热。目前未找到匹配的非处方药品，建议您咨询医生。请问症状识别是否正确？"

## 注意事项
1. 保持简洁，不超过3句话
2. 使用友好的医疗专业语气
3. 明确询问用户是否确认症状识别
